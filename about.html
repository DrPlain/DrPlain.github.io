SigmaCodes data analysis and visualization services

This project is my portfolio project built as a student of Alx. SigmaCodes is all about helping our clients tell the stories behind their data using interactive visualizations. Our web services offers our users the ability to upload their data in csv or excel format and interactively generates their desired charts. It also ensures that the appropriate charts are used for visualizations hence enabling those without statistical knowledge to still use our services.

Our team of experienced researchers are available to offer professional writing services, data analysis and consultation. Other services we offer include professional review of research papers, data cleaning services, questionnaire structuring and lots more.

Motivation for this project

My passion for data science prompted me find a way to automate data visualization processes such that those without indepth statistical knowledge can easily and accurately see the stories behind their data using interactive charts. Also, my love for backend web development informed my choice of using the django framework for this project.

Technologies, languages and major packages

  • Django python framework
  • Html
  • Bootstrap
  • Css
  • Pandas
  • Matplotlib
  • GIT for version control
  • Nginx + Gunicorn for deployment
